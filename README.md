# EchOS > Developed by geddada devicharan (@imdvichrn) & Echoless

A polished, offline-first study workspace built around neural syllabus mapping, deep focus, and personal progress insights.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Create a local-only `.env.local` file and add any secrets required by your environment.
3. Run the app:
   `npm run dev`

## Notes

- Do not commit `.env.local` or real API keys.
- `GEMINI_API_KEY` should stay in a local-only environment file if used.
- The build output lives in `dist/`.
